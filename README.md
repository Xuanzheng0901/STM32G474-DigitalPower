# DAB — 串联谐振双有源桥三移相控制

基于 STM32G474RET6 的隔离型双向 DC-DC 变换器，采用**三移相 (Triple Phase Shift, TPS)** 控制策略实现全范围软开关。

## 拓扑与原理

- 拓扑：串联谐振双有源桥 (SR-DAB)
- 变压器匝比：10:3 (n ≈ 3.33)
- 谐振频率：85kHz
- 开关频率范围：80kHz ~ 250kHz (线性调频)
- 控制自由度：αp (初级内移相) + αs (次级内移相) + θ (桥间移相)

### 三移相策略

| 控制量 | 作用 | 计算方式 |
|--------|------|----------|
| αp | 初级桥内移相，消除环流 | cos(αp) = M (M≤1 时) |
| αs | 次级桥内移相，消除环流 | cos(αs) = 1/M (M>1 时) |
| θ | 桥间移相，控制功率 | 前馈基准 + PID 动态调节 |

其中 M = n·Vlow/Vhigh 为电压增益。通过匹配 M 与 cos(α)，在降压/升压工况下均消除无功环流，提升效率。

## 工作模式

| 模式 | 功率方向 | 说明 |
|------|----------|------|
| SLEEP | — | 关闭全部 PWM 输出 |
| 1→2 | 正向 | 端口1 (10~15V) → 端口2 (3~4.2V)，预充电(500mA) + 可调恒流 |
| 2→1 | 反向 | 端口2 (3~4.2V) → 端口1 (10~15V)，预充电(100mA) + 可调恒流 |
| AUTO | 双向 | 自动维持端口1电压在 11~11.5V 范围内，双向功率自动调节 |

## 控制架构

```
ADC采样 → 卡尔曼滤波 → 电压增益M计算 → αp/αs前馈
                                    ↓
目标电流 ─→ PID (增量式) ─→ θ调节 → HRTIM_Update_Timing()
                                    ↓
                              5路Timer影子寄存器同步更新
```

- **PID 频率**: 500Hz (随 ADC DMA 周期触发)
- **PID 类型**: 增量式，统一油门值 [-0.5, 1.5]
- **前馈 + 闭环**: θ_ff 提供基础功率挡位，PID 做精调
- **滤波器**: 4 路独立一维卡尔曼滤波

## 硬件平台

- MCU: STM32G474RET6 (170MHz Cortex-M4)
- HRTIM: 5.44GHz，5 路 Timer (Master + A/B/C/D)
- ADC1/2: 差分采样电流，单端采样电压，过采样
- OLED: SH1107 128x64 (SPI)
- 编码器: 硬件定时器扫描
- RTOS: FreeRTOS + LVGL v9.3.0
